 Compañeros y colaboradores, en este fragmento de texto les hago mención sobre de que 
trata cada archivo dado que la estructura esta desordenada y por ende les comento. El trabajo 
consta de los objetivos puesto para cada semana dando como finalidad una propuesta de pagina 
web que permite realizar donaciones según proyecto de interés.
Readme para proyecto de GitHub Programación Web II Semana 8:
- Archivo S3 Web II, corresponde a la estructura de la página principal. 
- Archivo Evento, corresponde a la casilla para ingresar un evento u filtrarlo para la 
página de inicio. 
- Archivo S4, este tiene como fin conectar a la base de datos por lo tanto encontraras la 
validación a Mysql con la distribución de Maria DB (PhpMyAdmin). 
- S5, este corresponde a la pasarela de inicio de sesión de usuario, User: Admin / 
Password: 1234
- S5_1, este archivo corresponde a la página que se tiene predeterminada para llenado 
de datos de los usuarios a través de formulario.
- Archivo S6, pagina predeterminada para realizar la donación, posterior al llenado de 
información del S5_1 te redirige hasta aquí para finalizar la sesión.
- Archivo db, archivo de validación para bases de datos contiene consultas para enviar 
los formularios a sus correspondientes tablas y recuperación de datos. 
- Cerrar sesión, este documento contiene indexación para el término de la sesión de 
forma segura. 
- Style WebII, contiene codificación para diseño y formato de los distintos elementos 
dentro de esta página. 
- Webii, documento con script que contiene función para el filtrado de evento (archivo 
evento) y función para notificación de monto alcanzado de donación de pagina 
principal.
